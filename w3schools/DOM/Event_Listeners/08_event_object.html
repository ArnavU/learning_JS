The event Parameter:
The event parameter is an object that contains information about the event that occurred. It provides properties and methods that allow you to access details such as:

        event.type: The type of the event ('click', 'keydown', etc.).
        event.target: The element that triggered the event.
        event.currentTarget: The current element that has the event listener.
        event.preventDefault(): A method to prevent the default behavior of the event (e.g., prevent a form submission).
        event.stopPropagation(): A method to stop the event from propagating up or down the DOM tree.
